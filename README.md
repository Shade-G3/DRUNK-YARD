# Drunk Yard
Drunk Yard – Real-Time Anonymous Video Chat Platform

Drunk Yard is a real-time anonymous video chat application designed to connect users based on shared preferences in a fun and engaging way. Inspired by platforms like Chatroulette and Omegle, this project introduces a unique twist by allowing users to match based on selected “drink vibes” such as Whisky, Rum, Vodka, or Wine—creating a more personalized and interest-driven interaction.

The platform is built using modern web technologies with a focus on real-time communication, seamless user experience, and scalable architecture. It leverages WebRTC for peer-to-peer video streaming, enabling low-latency, high-quality video communication directly between users without relying on heavy server bandwidth. Socket.io is used as the signaling mechanism to handle matchmaking, connection setup, and real-time messaging between users.

The application features a dynamic matchmaking system where users are placed into queues based on their selected category. When two users choose the same option, they are instantly connected via video and can also communicate through a live chat interface. The UI is designed with a modern dark theme, smooth transitions, and responsive layout to enhance usability and provide a clean, intuitive experience.

Key features include:

Category-based matchmaking (interest-driven connections)
Real-time video communication using WebRTC
Integrated live chat system
Smooth UI/UX with animated transitions
“Next” (skip) functionality for quick rematching
Queue management and real-time user handling

Drunk Yard demonstrates a strong understanding of full-stack development, including frontend design, backend architecture, real-time systems, and debugging complex issues such as WebRTC signaling, socket communication, and UI layering.

This project is a step toward building scalable, real-time social platforms and showcases practical implementation of networking concepts, event-driven programming, and user-centric design.
